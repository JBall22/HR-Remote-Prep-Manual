Getting Started

Welcome! This is meant to be a staging area for your fight to get into Hack Reactor. With links aplenty to valuable resources, it should prove to be invaluable in your quest. First off, this is a video meant to explain what exactly it means to want to learn to code by a guy who goes by the handle CheersKevin on Youtube.



What is Hack Reactor?

Hack Reactor, in their own words, is: "Not your typical programming, coding or development bootcamp, Hack Reactor is an elite software career accelerator with unmatched results."

Their goal: "You’ll begin Hack Reactor with a feeling of excitement and anticipation. Twelve weeks later, you’ll follow the footsteps of our trailblazing alumni, taking the methodologies and best practices you perfected at our coding bootcamp to your next job. We’ve built world class software engineering curriculum and programming courses. However, Hack Reactor is, above all else, a world-class learning environment."

Our goal: To provide information that you can use to prepare yourself for the application, interview, and main course of Hack Reactor. For more information on the school itself check out the following links.



Hack Reactor Prep Courses

Hack Reactor does offer their own preparatory classes. These are paid classes, though you have the option of putting the admission fees towards the full immersive programs if you choose to attend any of their courses.

  • For beginners to coding there is Hack Reactor Remote Prep.

    • The course mixes lectures with solo coding and pair programming. This gives students a guided experience that allows them to establish an understanding of fundamentals through experience and by learning from each other. Pair programming has the added benefit of teaching students to communicate about code, which is a key skill in the professional world. Instructors circulate among students throughout the coding portions of the class to help students out of ruts and to iron out concepts. Class sizes are limited to 20 to allow for plenty of individual instruction.

  • For more advanced users there is Fulcrum.

    • Fulcrum is the most comprehensive self-learning curriculum on JavaScript development, containing over 100 hours of material.



Learn

Whether it be learning the basics, understanding the theory behind it, or getting a functional working knowledge of different technology types, this section should cover you. Provides a variety of options for you to choose from based on what you want to learn and what style of learning works best for you.

Command Line

Ever used your terminal before? Well get ready to!

  • Codecademy: Learn the Command Line

    • Gets you learning and trying out the basics of the command line.

  • Command Line Crash Course Book

    • Learn the command line the hard way. This book's primary goal is getting you proficient in the command line in a limited amount of time without holding your hand along the way.

Git

Git is the most popular version control system. It is a way to track your progress through a project, and to have others collaborate with you.

  • GitHub

    • The most popular web-based Git repository hosting service, and a great starting point.

  • GitHub - Good Resources for Learning Git and GitHub

    • A great list of other resources for learning Git and GitHub

  • Code School - Discover Git

    • A free 25 step course that puts you at the helm of Git through terminal commands.

HTML/CSS

The nouns and adjectives of the web!

  • Codecademy: HTML and CSS

    • Learn while doing with the basics of HTML and CSS. Don't get too attached to the training wheels that accompany these Codeacademy courses, as they will be gone when you get into the reality of programming.

  • Free Code Camp

    • Much like the style of Codecademy, you learn by doing with their interactive online experience.

  • Shay Howe: Learn to Code HTML and CSS

    • A simple and comprehensive guide dedicated to helping beginners learn HTML and CSS. Outlining the fundamentals, this guide works through all common elements of front-end design and development. Also, has an Advanced Guide for those looking to take a deeper look at front-end design and development, perfect for any designer or front-end developer looking to round out their skills.

Bootstrap

The most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

  • Bootstrap Getting Started

    • Their own walkthrough getting started. Caution: May be a little intimidating if you are new to coding.

  • Free Code Camp

    • Once again FreeCodeCamp is here for you with a follow along Bootstrap section.

  • Stack Overflow - Bootstrap Resources

    • A great beginners Bootstrap resource

JavaScript

Time for some verbs.

  • Codecademy: Learn JavaScript

    • Become familiar with the basics, by writing along with Codecademy

  • Learn JavaScript in Y Minutes

    • Learn JavaScript by reading through actual code with comments.

  • Free Code Camp

    • Useful for getting a good base in all things JavaScript related, while also getting some experience solving problems on your own and creating your own projects.

  • Code School

    • Highly recommended by many alumni is the JavaScript Road Trip on Code School.

  • Eloquent JavaScript Book: First 5 Chapters

    • Great for helping you to understand the theory behind JavaScript. (More useful if you have some prior experience with JavaScript)

  • JavaScript.isSexy

    • Regarded as essential reading from most alumni. This is a vital resource for many things JavaScript.

jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

  • Getting Started with jQuery UI
  • Quick API Reference

    • Invaluable resource on jQuery syntax lines.

Tools

Here are some of the tools of the trade. These will help get you started in the world of programming and help you to set up your development environment the way you like it.

Programming-Specific Text Editors: Choose your Weapon

Sublime Text

Free full-featured unlimited trial, but will bug you every certain number of saves to buy the full version for $70. Runs fast and you can download some nice packages to make your life easier. Currently the preferred text editor by programmers, but Atom is catching up fast.

Useful Keyboard Shortcuts for Sublime
Packages

Directories that can be installed into your text editor to give it additional functionalities, customization, or quality of life changes.

  • Package Control

    • The Sublime Text package manager that makes it exceedingly simple to find, install and keep packages up-to-date. After installing this through the Sublime Text console (ctrl+`), press ctl+shift+p (Windows/Linux) or ⇧+⌘+p (OSX) to open up command menu and go to 'Package Control: Install Package' to install other packages.

  • Colorsublime

    • A theme manager that lets you preview/install themes directly through Sublime Text from the Colorsublime Repository.

  • Emmett

    • Emmet is a web-developer’s toolkit that can greatly improve your HTML and CSS workflow. Takes snippets to a new level and lets you use your tab key after typing commands to do a number of additional useful shortcuts. Here is a handy cheat sheet you can use on the fly to help get you started.

Atom

Free, Open Source, text editor by Github that is relatively new to the scene but rising fast. Easily customizable and user-friendly, and integrates well with Git. Runs a bit more sluggish than alternatives, but makes up for that in other areas.

Packages

Can be accessed by opening the preferences menu under Atom and navigating to packages or themes to see current ones, and install to get new ones. No need for package control or theme managers.

  • Emmett

    • This is a ported over version of Sublime Text's Emmett package for Atom, and it contains most of the same functionalities. Again, a useful cheat sheet to get you started.

  • Minimap

    • For those missing the minimap feature from Sublime Text, there is no need to fear. Minimap package is here.

  • Power Mode

    • If you are finding the coding process a little mundane, then you can activate Power Mode! This will make you feel like you are writing code in an action movie, and also possibly annoy you a little or a lot with time. Kingdoms have been raised and been brought down with the use of Power Mode, so use with care.

Communication

Slack

Messaging tool used by many teams. It's easy to use and has many features essential to programmers.

Floobits

Pair programming is a great tool for learning to code, and Floobits lets you do it remotely, right through your text editor. First you'll need to sign up on their website(You can use your Github account) then set it up in your text editor: Atom | Sublime

Accounts, Programs, and Plugins

Github

Github is a website that allows you to easily share git repositories. Creating an account is important as it can serve as a portfolio and store all your projects, while letting you work on them together with others as well.

XCode Command Line Tools (Mac only)

This is a requirement for many common programming tools.

Homebrew (Mac only)

Homebrew is a package manager - that basically means it handles the proper installation and configuration of many programs you'll use as a software developer. The following programs can be easily installed with it.

  • Git Version Control

    • The simplest way to install git is to use Homebrew by running this command in the terminal |brew install git| (Mac only)

  • Node.js and NPM: the LTS download is best for now

    • Node is a way to run JavaScript code outside of a web browser, while NPM is just a package manager for JavaScript programs and comes bundled along with the node download.
    Again, you can also just install node using Homebrew by running the command |brew install node| in the terminal. (Mac only)
